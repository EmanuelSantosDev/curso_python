# Conversão de Tipos

````python
a = '3'

# conversão para Inteiro
b = int(a)

# conversão para Float
c = float(b)

# conversão para String
d = str(c)
````

### Conversão Automática

````python
a = 10
b = 5

print(type(a))  # <class 'int'>
print(type(b))  # <class 'int'>

resultado_divisao = a / b

print(type(resultado_divisao))  # <class 'float'>
````
