# Operadores Relacionais


### Comparações Numéricas
```python
idade = 15
print(idade > 17)  # False
print(idade < 17)  # True
print(idade <= 17)  # True
print(idade >= 17)  # False
print(idade == 17)  # False
print(idade != 17)  # True
```

### Comparações Entre Outros Tipos
```python
print(True == True)  # True
print('Rafael' == 'rafael')  # False
print('b' > 'a')  # True
print(5 == '5')  # False
```
